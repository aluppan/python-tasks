#1 Пользователь вводит N-ое количество чисел. Ввод чисел прекращается когда пользователь пишет stop. Среди всех чисел найдите:
- Сумму чётных чисел
- Произведение нечётных чисел
- Среднее арифметическое для каждой группы выше (т.е среднее арифм. для чётных чисел и среднее арифм. для нечётных)

#2 Календарь Арпеля 2021 года. Пользователь вводит день для текущего месяца. Для каждого числа выведите день недели. Например,
27 - вторник,  5 - понедельник. Месяц и год из которого должны получаться значения: Арпель, 2021 год. Получение дня недели не должно быть одноразовым (т.е пользователь может ввести N-ое количество раз число месяца Апреля 2021 года и получить день недели).
